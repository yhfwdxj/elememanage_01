# 关于本项目
## 简介
本项目是[仿饿了么小程序](https://github.com/yhfwdxj/eleme_01_uniapp)的后台管理系统，接口数据来源于[node-elm](https://github.com/bailicangdu/node-elm/blob/master/API.md) 
### 技术栈
vue3 + vue-router + vuex + axios + sass+ elementUI-plus
## 项目启动
```
yarn add 
yarn dev
```
